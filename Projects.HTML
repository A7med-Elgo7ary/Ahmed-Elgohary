Smart Agriculture Greenhouse System Using IOT Technology

Our project introduces an IoT-based intelligent greenhouse system, developed as my graduation project for a Bachelor's degree, designed to optimize crop growth, reduce environmental impact, and support sustainable agriculture. It integrates **Arduino Nano 33 IoT** with advanced sensors and actuators to monitor and control environmental conditions like temperature, humidity, soil moisture, and light. A **mobile app** provides role-based access (Admin, Farmer, Visitor) for real-time monitoring and control, while **AI** analyzes sensor data to predict optimal growth conditions and detect plant diseases. The system ensures robust **IoT security** through TOTP, RC4 encryption, and MD5 hashing, and leverages **Firebase** for cloud-based management, enabling scalability and remote access. This innovation aims to enhance agricultural productivity, promote sustainability, and demonstrate the practical application of IoT and AI technologies in modern farming.

Key Features

  • Automated Climate and Soil Optimization System
  • Multi-Sensor Environmental Monitoring Network
  • AI-Driven Growth Analytics and Disease Prediction
  • Role-Based Mobile Control Interface Platform
  • Encrypted IoT Data Transmission Protocols
  • Cloud-Based Remote Management via Firebase

Technologies Used

Arduino Nano 33 IoT Advanced Sensors Actuators Mobile Application Artificial Intelligence IoT Security Protocols Cloud Platform (Firebase) Wireless Communication
